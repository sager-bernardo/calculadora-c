# Calculadora em C

Implementação de uma calculadora baseada em texto, desenvolvida em C como projeto da plataforma Neps Academy.

## Funcionalidades
- Operações aritméticas básicas via terminal
- Leitura e processamento de entrada do usuário

## Tecnologias
- Linguagem C
- Interface por terminal
